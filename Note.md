## Status codes and HTTP methods

- GET   : retrive an existing resource (read only)

- PUT   : update an existing resource
- PATCH : partically update an existing resoure
- DELETE: delete a resource

HTTP status code
- 2xx   : successful
- 3xx   : redirect
- 4xx   : client error
- 5xx   : server error
